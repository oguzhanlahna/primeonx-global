# Primeonx Global Website (Portfolio Version)

This is a curated portfolio-safe version of the Primeonx corporate website project built with ASP.NET Web Forms.

## Included
- Homepage and core marketing pages
- Master page and shared layout
- Blog and case study pages
- Contact and policy pages
- App_Start configuration
- Required assets for presentation

## Removed for public sharing
- Real credentials, connection strings, SMTP settings
- Build output and local IDE files
- Test SMTP pages and non-essential internal files

## Stack
- ASP.NET Web Forms
- C#
- SQL Server
- HTML/CSS/JavaScript
